Lệnh else if trong js

sử dụng khi có nhiều hơn 1 điều kiện

        Cú pháp lệnh if else if else

        if(điều kiện 1){
            //code khi đk 1 true
        }else if(điều kiện 2){
            //code khi đk 1 false và đk 2 true
        }else if(điều kiện 3){
            //code khi đk 1,2 false và đk 3 true
        }else {
            //code khi đk 1,2,3 false
        }
