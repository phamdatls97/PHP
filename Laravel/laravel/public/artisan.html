Các câu lệnh artisan trong laravel
        -Để di chuyển trong thư mục thì dùng câu lệnh cd
        VD: cd tên thư mục
        nếu muốn quay ngược ra ngoài 1 folder thì gõ cd..
        -Artisan là công cụ các câu lệnh cho phép tạo nhanh các chức năng trong laravel
        nhu là: tạo controller, tạo model, tạo seeder, tạo migrate, tạo lại app key bảo mật.
        -Để chạy được câu lệnh artisan trong laravel
        thì cần truy cập vào thư mục gốc của ứng dụng (thư mục mẹ trực tiếp chưa file .env)
        hoặc (thư mục mẹ trực tiếp chưa file artisan.php)
        -Câu lệnh đầu tiên:
        1 . PHP artisan
        câu lệnh sẽ hiển thị tất cả các câu lệnh artisan kèm theo ý nghĩa của từng câu lệnh đó
        2 . Tạo ra 1 controller trong laravel
        php artisan make:controller Test123Controller
        (Lưu ý nếu chỉ gõ trên controller thì controller đó sẽ tạo trực tiếp trong folder
        app/Http/Controllers )
        3 . Trong 1 hệ thống phức tạp thì controller sẽ đc chia vào nhiều folder khác nhau nếu muốn tạo
        controller nằm trong 1 đường dẫn folder thì gõ:
        php artisan make:controller Backend/ProductController
        php artisan make:controller Frontend/ProductController
        4 . Tạo model trong Laravel
        Model trong laravel mặc định nằm trực tiếp trong app
        Model mặc định là User.php trong folder app
        - Tạo ra 1 folder chứa model "Models" và 1 model trực tiếp trong đó
        => php artisan make:model Models/TestModel
        - Tạo ra 1 model nằm trong thư mục cấp 2 của Models
        php artisan make:model Models/Backend/ProductModel
        php artisan make:model Models/Frontend/ProductModel.
        5 . Tạo migration (file dùng để tạo table trong CSDL mà không cần tương tác với navicat hoặc phpmyadmin)
        php artisan make:migration create_tên bảng_table
        => php artisan make:migration create_product_table
        sau khi chạy xong câu lệnh trên vào folder database/migrations
        sẽ thấy 1 file php có định dạng "thời_gian_tên_migration" vừa tạo
        php artisan make:migration create_category_table.
        6 . Sau khi migration và cấu hình đúng tên CSDL đã tồn tại trong .env
        => tạo từ migration thành bảng CSDL thật
        thì chạy: php artisan migrate
        qtrinh chạy lần đầu sẽ gặp lỗi.
