Margin: auto

Auto đc sử dụng để căn 1 thẻ html có độ rộng xác định như là 100px,200px,....vào giữa màn hình

All the margin properties can have the following values: auto - the browser calculates the margin length - specifies a margin in px, pt, cm, etc. % - specifies a margin in % of the width of the containing element inherit
