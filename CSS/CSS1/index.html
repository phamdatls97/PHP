CSS Syntax ( cú pháp ) and selector ( Bộ chọn )

A CSS rule-set consists of a selector and a declaration block:

Một quy tắc cú pháp css bao gồm 1 selector và 1 khối khai báo code

The selector points to the HTML element you want to style

Selector sẽ chỉ ra thẻ html mà bạn muốn trang trí

The declaration block contains one or more declarations separated by semicolons.

một đoạn khai báo chứa 1 hay nhiều đoạn khai báo mã css phân tách bởi dấu;

Each declaration includes a CSS property name and a value, separated by a colon

Mỗi đoạn khai báo bao gồm tên và giá trị của 1 thuộc tính css đc phân tách bởi dấu :

A CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces

Một khai báo css thường xuyên kết thúc với dấu ; và khối khai báo đc bao quanh bởi {}

Ví dụ về code css : 

        p {
          color: red;
          text-align: center;
        }
